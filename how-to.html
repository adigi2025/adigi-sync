How To use Adigi Sync

Privacy-first, in-browser conversion. No uploads.

  1. Open the tool (GitHub Pages URL or index.html locally).
  2. Load a file: drag & drop a CSV / XLSX / JSON or click Browse.
  3. Adjust options:
    • Target format: CSV, XLSX, or JSON
    • CSV delimiter (e.g. , or ;)
    • Header row, Normalise columns, Deduplicate rows
  4. Preview the first rows or JSON before download.
  5. Click Convert & Download to save the converted file.

Troubleshooting

  • Seeing raw text? Hard-refresh (Cmd+Shift+R).
  • CSV delimiters vary; set the right one before converting.
  • Large XLSX files: try a fresh browser tab; it’s all local memory.
